# API backend Express.js Mongoose
To start developing run `npm install` and then `npm run start:nodemon`

Instructions:

```
npm i
```

Setup Mongo DB:
```
docker run --name mongodb -d -p 27017:27017 mongo
```

```
npm run dev
```
